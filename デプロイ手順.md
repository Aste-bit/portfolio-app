# ポートフォリオアプリ デプロイ手順

## Claude Codeに渡すプロンプト

以下をClaude Codeにそのまま渡せばOKです:

```
このportfolio-appフォルダをGitHubにpushして、Vercelにデプロイしてください。

手順:
1. git init して全ファイルをコミット
2. GitHubにPublicリポジトリ「portfolio」を作成（gh CLIまたは手動）
3. git remote add origin して push
4. Vercelにログインしてリポジトリをimport → デプロイ

注意:
- node_modules/ は .gitignore に入っているのでpushされません
- Vercelは自動でビルド設定を検出します（Framework: Vite）
- 環境変数は不要です（外部API接続なし）
```

## 手動でやる場合の手順

### 1. GitHubリポジトリ作成
- GitHub.com で新しいリポジトリを作成
- リポジトリ名: `portfolio`（好きな名前でOK）
- Public にする（ポートフォリオなので公開）

### 2. ターミナルで実行
```bash
cd portfolio-app
git init
git add .
git commit -m "initial commit: portfolio app"
git remote add origin https://github.com/あなたのユーザー名/portfolio.git
git push -u origin main
```

### 3. Vercelでデプロイ
1. https://vercel.com にアクセス（GitHubアカウントでログイン）
2. 「Import Project」→ GitHubのportfolioリポジトリを選択
3. そのまま「Deploy」をクリック
4. 1分程度でデプロイ完了 → URLが発行される

### 4. 完了
`https://portfolio-xxxxx.vercel.app` のようなURLが発行されます。
このURLを共有すればポートフォリオが見られます。

## 更新する時
ファイルを編集して `git push` するだけで自動的にVercelが再デプロイします。
